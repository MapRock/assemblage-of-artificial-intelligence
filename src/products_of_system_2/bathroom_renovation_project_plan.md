# Bathroom Renovation Tasks with Decisions and Dependencies

## 1. Planning and Design

**Tasks**
- Measure bathroom
- Sketch layout
- Choose fixtures (toilet, vanity, tub/shower)
- Choose tile and flooring
- Establish budget
- Order materials

**Dependencies**
- None (starting phase)

**Decisions**
- Keep existing layout or move plumbing?
- Shower, tub, or tub/shower combo?
- Single vanity or double vanity?
- Tile floor vs vinyl vs stone?

**Implications**
Moving plumbing or walls increases cost and may require permits.

---

## 2. Permits and Preparation

**Tasks**
- Determine permit requirements
- Submit permit application if needed
- Order long-lead items (tile, vanity, fixtures)
- Clear room
- Protect surrounding areas

**Dependencies**
- Design decisions complete

**Decisions**
- DIY vs contractor
- Permit or no permit (depends on jurisdiction)

---

## 3. Demolition

**Tasks**
- Remove vanity
- Remove toilet
- Remove tub or shower
- Remove tile
- Remove flooring
- Remove drywall in wet areas

**Dependencies**
- Materials ordered
- Water shut off

**Decisions**
- Partial demo or full gut renovation?

**Possible discoveries**
- Mold
- Water damage
- Rot in framing
- Old plumbing

These discoveries may introduce additional tasks.

---

## 4. Structural Repair (conditional)

**Tasks**
- Replace damaged studs
- Reinforce floor if needed
- Replace subfloor

- Demolition complete

## 5. Rough Plumbing

**Tasks**
- Install new supply lines
- Install drain lines
- Install shower valve
- Relocate plumbing if layout changed

**Dependencies**
- Demolition complete
- Framing accessible

**Decisions**
- Move fixtures?
- Upgrade to modern plumbing (PEX, etc.)?

---

## 6. Rough Electrical

**Tasks**
- Install new wiring
- Install GFCI outlets
- Install lighting boxes
- Install exhaust fan

**Dependencies**
- Walls open
- Layout finalized

**Decisions**
- Add heated floor?
- Add recessed lighting?
- Upgrade fan ventilation?

---

## 7. Inspection (if permits required)

**Tasks**
- Rough plumbing inspection
- Rough electrical inspection

**Dependencies**
- Rough work complete

---

## 8. Wallboard and Waterproofing

**Tasks**
- Install cement backer board
- Install drywall outside wet areas
- Install waterproof membrane
- Install shower pan or tub

**Dependencies**
- Plumbing and electrical rough-in complete

**Decisions**
- Waterproofing method (membrane, RedGard, Schluter)

---

## 9. Flooring and Tile

**Tasks**
- Install floor tile
- Install shower tile
- Install accent tile
- Grout tile
- Seal grout if required

**Dependencies**
- Waterproofing completed

**Decisions**
- Tile pattern
- Niche placement
- Heated floor installation

---

## 10. Fixtures Installation

**Tasks**
- Install vanity
- Install sink and faucet
- Install shower fixtures

- Install toilet

**Dependencies**

## 11. Electrical Finish
